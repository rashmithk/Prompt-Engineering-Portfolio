# Install Google Chrome

## Objective

Generate a PowerShell script to silently install Google Chrome.

## Prompt

You are a Senior Windows Administrator.

Generate a PowerShell script that:

- Downloads the latest Chrome installer
- Installs silently
- Supports Endpoint Central deployment
- Creates a log file
- Handles errors
- Returns appropriate exit codes

Output only the PowerShell script.

## Example Input

Install Google Chrome silently.

## Expected Output

A production-ready PowerShell script.

## Lessons Learned

Adding deployment requirements and output constraints produced more reliable results.
